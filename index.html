Como Usar
📌 Adicionando um novo relatório
  1. Gere o relatório do dia com os dados da planilha
  2. Salve o arquivo no padrão: relatorio_DDMMM.html (ex: relatorio_26maio.html)
  3. Faça upload no GitHub via Add file → Upload files
  4. Adicione o card correspondente nesta página no array relatorios abaixo
  5. Faça commit — o portal atualiza automaticamente
